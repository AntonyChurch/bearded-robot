Bearded-Robot

Use bearded-robot to write GPIO controlling things with Erlang on your Raspberry Pi. It is not currently perfect, but it does the job of turning ports both on and off. There is still a long way to go however, I only add to the project when I need an additional feature. Please feel free to jump in and help out!

Requirements

  1. Erlang Runtime
  2. Wiring Pi
